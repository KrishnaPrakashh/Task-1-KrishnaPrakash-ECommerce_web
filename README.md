# Project 1 - E-Commerce Website

## Overview

A responsive E-Commerce website developed using Angular as part of the DecodeLabs Internship Program. The application displays products from an API and provides shopping features such as cart and wishlist management.

## Features

* Responsive Design
* Product Listing
* Product Details
* Shopping Cart
* Wishlist
* Login Page
* API Integration
* User-Friendly Interface

## Technologies Used

* Angular
* TypeScript
* HTML
* CSS
* Bootstrap

## Installation

```bash
npm install
```

## Run the Project

```bash
ng serve
```

Open:

```text
http://localhost:4200
```

## Build the Project

```bash
ng build
```
